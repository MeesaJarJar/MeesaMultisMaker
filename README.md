# MeesaMultisMaker
Meesa Multis Maker is a Windows tool by MeesaJarJar for creating and editing Ultima Online multi structures and GUMP graphics. Features an isometric canvas with drag-and-drop placement, transforms (scale/rotate/flip/skew), AI-powered texture regeneration via ComfyUI, MUL/UOP file support, and a built-in pixel painter.
